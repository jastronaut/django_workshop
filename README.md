# django_workshop
a brief introduction to django, for icssc @ uci

## Dependencies
- Python 3, pip3
- A text editor
- A terminal emulator
- Git

## Installing Django

- Run `pip3 install Django==2.0.5` in your terminal
- Check it's installed with `python3 -m django --version`
